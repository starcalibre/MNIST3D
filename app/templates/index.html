MNIST Dataset 3D

The MNIST dataset is a collection of 60,000 handwritten digits ranging from 0-9. This visualisation takes a random sample of 10,000 digits and maps them to a three dimensional space using the very cool method of TSNE. TSNE has grouped each image representing the different digits quite nicely.

Explore the visualisation by zooming with the mouse wheel, panning by dragging with the right mouse button and rotating by dragging with the left mouse button. Clicking on points will display the corresponding image in the above panel.

The visualisation is built using the excellent Three.js library.
